# 合同财务风险审查框架

5 大财务风险维度 + 量化检查清单。

## 维度 1：付款条款审查

- 付款节奏（预付/里程碑/尾付比例是否合理）
- 账期（交付到付款的周期对现金流的影响）
- 付款条件（是否挂钩验收/里程碑/发票）
- 付款方式（电汇/承兑/信用证 — 承兑增加隐性成本）
- 质保金/保证金（比例、退还条件、占用成本）

## 维度 2：税务审查

- 合同涉及的税种（增值税/所得税/印花税 — 逐项确认）
- 税率与含税/不含税约定
- 发票类型与开具时点
- 跨境税务（预提所得税、税收协定、转移定价）
- 税务发票与付款的匹配性

## 维度 3：成本完整性

- 合同总对价 vs 实际总成本（含隐藏成本）
- 汇率风险（外币合同 — 锁汇/自然对冲/敞口）
- 第三方费用承担（审计费、评估费、登记费、中介费）
- 违约金计算基数（合同总额 vs 未履行部分 vs 实际损失）
- 通胀/价格调整机制（长期合同是否有调价条款）

## 维度 4：违约金与赔偿

- 违约金比例是否合理（《民法典》585 条 — 超损失 30% 可调整）
- 赔偿上限是否有封顶（总限额 vs 单次限额）
- 间接损失是否排除（利润损失、商誉损失）
- 累计违约金是否有上限（避免叠加无限放大）

## 维度 5：财务可审计性

- 条款是否清晰可入账（收入确认时点、成本归集）
- 可审计性（凭证链是否完整、条款是否支持审计取证）
- 可对账性（结算周期、对账机制、差异处理）
- 预算影响（是否超出预算、是否需要追加预算）

## 量化检查清单

- [ ] 合同总金额（含税/不含税）
- [ ] 付款节奏（预付 % / 里程碑 % / 尾付 %）
- [ ] 账期天数（交付 → 验收 → 付款的天数）
- [ ] 质保金比例 + 退还条件
- [ ] 涉及税种 + 税率
- [ ] 违约金比例（占合同总额 %）
- [ ] 违约金是否有上限
- [ ] 汇率敞口（如外币）
- [ ] 隐藏成本总额估算
